# Carioca Craft Store

Loja visual completa para servidor Minecraft.

## Importante
Esta versão está pronta para publicar como site visual e painel funcional no navegador usando localStorage.
Para pagamentos reais Mercado Pago, banco real e entrega automática, depois precisa conectar backend + banco.

## Como usar no GitHub/Railway
Suba tudo no GitHub. Se quiser hospedar apenas o visual, use a pasta `frontend`.

Admin padrão:
- Email: admin@cariocacraft.com
- Senha: admin123

Troque isso depois no arquivo `frontend/js/app.js`.
