# Loja Minecraft Profissional

Backend Node.js + PostgreSQL/Supabase + frontend HTML/CSS/JS.

## Como usar
1. Suba para o GitHub.
2. Crie o banco no Supabase e rode database/schema.sql.
3. Hospede o backend no Render usando rootDir backend.
4. Configure as variáveis de ambiente no Render.
5. No frontend/js/api.js, troque API_URL pela URL do Render.
